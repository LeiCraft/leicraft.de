# Linus Website

My personal portfolio built with **Nuxt 4**, **Nuxt UI 4** and **Tailwind CSS 4**.

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun run dev
```

### Compile and Minify for Production

```sh
bun run build
```

### Generate Static Site

```sh
bun run generate
```
